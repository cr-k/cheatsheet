HTML and CSS Cheatsheet

HTML Basic Tags


What is an HTML element?

An HTML element is a piece of content in an HTML document and uses the following syntax:
opening tag + content + closing tag

What is a tag?

HTML tags are keywords within a webpage that define how your browser page is formatted.
Example: <div>


Tag/Attribute Name Description
<div> Division Divides document into sections. Good for class or id attribute.
<h1> - <h6> Heading HTML can support six different levels of headings. The largest is h1 and the smallest is h6.
<img>* Image The image element embeds images in documents. Needs to have src="" defined.
<a> Anchor Used to create hyperlinks in an HTML document. Needs to have src="" defined.
<ol> Ordered List Creates a list of items by numbering them. Used with <li>.
<ul> Unordered List Creates a list of items by using bullet points. Used with <li>.
<br> Line Break Produces a line break in text.
<hr> Horizontal Rule Produces a horizontal line between elements.

*Self-closing tag



HTML Table Tags


Tag/Attribute Name Description
<table> Table The element that is used to represent data with a two-dimensional table made of rows and columns.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows that contain actual table data.
<th> Table Heading Adds titles to rows and columns of a table. Must be enclosed in a <tr>.
<tr> Table Row Represents a row of cells in a table.
<td> Table Data The data inside of a table row.
<tfoot> Table Foot The table rows that define a footer at the end of a table.
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.



CSS Selectors and Specificity


What is CSS used for?

CSS is used to describe how a webpage should be presented. Without CSS, and only using HTML, a website would look plain and boring.

What can you change with CSS?

Listed in properties, you can change most things about how a webpage displays.


Selector Example Description
.class .intro Selects all elements with class="intro".
.class1 .class2 .rectangle1 .rectangle2 Selects all elements with both rectangle1 and rectangle2 within its class attribute.
.class p .intro p Selects all <p> elements that are descendants of an element with class="intro".
#id #title Selects all elements with id="title".
element p Selects all <p> elements.
element .class p .intro Selects all <p> elements with class="intro".
element, element div, p Selects <div> and <p> elements.
element element div p Selects all <p> elements inside of <div> elements.



Common CSS Properties


Property Description
background-image Defines an element's background image.
border Sets the width, style, and color for all four sides of an element's border.
border-collapse Specify whether table cell borders are connected or seperated.
color Specify the text color of an element.
column-span Specifies how many columns an element spans in a multi-column layout.
display Specifies how an element displays on-screen.
font-family Defines a list of fonts for an element.
font-size Defines the font size of the text.
font-weight Specify the font weight of the text.
letter-spacing Sets extra spacing between letters.
line-height Sets the line height between text.
list-style-type Specifies the marker style for a list-item.
margin Sets the margin on all four sides of an element.
max-height Specify the maximum height of an element.
opacity Specifies the transparency of an element.
padding Sets the padding on all four sides of an element.
position Specifies how an element is positioned.
text-align Sets the horizontal alignment of inline content.
text-decoration Specifies the decoration added to text.
text-transform Transforms the case of the text.
white-space Specifies how the white space in an element is handled.
width Specify the width of an element.
word-spacing Sets the spacing between words.
word-wrap Specifies whether to break words when the content overflows the boundaries of its container.
